- Knowledge of C++
- Basics of OOP (Object-Oriented Programming)
- Main features of OOP:
  - Abstraction
  - Encapsulation
  - Inheritance
  - Polymorphism
  - File I/O operation
  - Constructor
  - Operator overloading
  - Function overloading
  - Friend function
  - virtual keyword to resolve Diamond shape problem
